# 撲克牌遊戲 (Poker Game)

Windows Forms 撲克牌換牌遊戲，支援發牌、換牌、牌型判斷與下注功能。

## 功能特色

- 隨機發 5 張牌
- 點擊牌面選擇要換的牌（可換任意張）
- 判斷牌型（皇家同花順、同花順、四條、葫蘆、同花、順子、三條、兩對、一對、雜牌）
- 下注系統：設定押注金額，依牌型賠率計算獎金
- 鍵盤秘技：按 Q/W/E/R/T/Y 可快速測試特定牌型

## 執行說明

1. 使用 Visual Studio 2022 開啟 `Poker.sln`
2. 建置專案（Build → Build Solution）
3. 執行（F5）

## 操作說明

1. 選擇下注金額，點擊「下注」
2. 點擊「發牌」取得 5 張牌
3. 點擊牌面可切換是否更換該張牌（牌面向下表示要換）
4. 點擊「換牌」更換選中的牌
5. 點擊「判斷牌型」查看結果與獎金

## 開發環境

- Visual Studio 2022
- .NET Framework 4.8
- C# Windows Forms
